## 📝 고객 요구사항 정리
금액을 정확히 계산하여 보여주기
총 금액이 10만원 미만이라면 배송비 3000원, 10만원 이상은 배송비 무료
뒤로가기 버튼을 이용해 메인 상품 목록 페이지로 이동
결제하기 버튼을 누르면, 결제 모듈을 모달로 사용할 수 있게 연동

## 📋 기능 목록
뒤로가기 버튼, 상품 수량, 배송비, 계산 영역, 결제 기능 연동
